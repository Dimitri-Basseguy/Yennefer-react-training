## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* Git
```sh
sudo apt-get install git
```
[product-screenshot]: /docs/todo-list-dimitri-basseguy-screen.png

### Installation
 
1. Clone the repo
```sh
git clone https://github.com/Dimitri-Basseguy/Todo-list-React
```
2. Open VS Code
3. Install dependencies
```
yarn
```
3. Launch the server
```
yarn start
```
